NullPoint Functions is a WordPress plugin for NullPoint theme from Webzakt.

== Description ==

NullPoint Functions is a WordPress plugin for NullPoint theme from Webzakt.
The features works only with Unyson plugin and with NullPoint theme.

NullPoint Functions contains unique page builder elements and some extra features for the theme.

If You like the demo of the NullPoint theme, You can download the content and import it to Your site with the NullPoint Functions plugin. The import demo tool will download content from our Webzakt.com server and install it onto your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/nullpoint-functions` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
